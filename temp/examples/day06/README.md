Day06 Demo Examples
====================

Quick runner for Day06 module demos. Examples are organized under:

- `module15/` — OOP & DataSource demos
- `module16/` — Iterators, Generators, Streaming pipeline
- `module17/` — Config-driven pipeline demos

Run examples from the repository root. Example commands:

Python path helper ensures imports work when executed as scripts.

Module 15:
```
python temp/examples/day06/module15/pipeline_v1.py
python temp/examples/day06/module15/pipeline_runner.py
python temp/examples/day06/module15/demo_incomplete_class.py
```

Module 16:
```
python temp/examples/day06/module16/pipeline/streaming_pipeline.py /tmp/demo_pipeline.log
```

Module 17:
```
python temp/examples/day06/module17/main.py
```
