# Python Keyword Reference

This quick reference lists Python keywords grouped by concept with one-line, beginner-friendly explanations.

## Control Flow Keywords
- if – Execute a block when a condition is true.
- elif – Test another condition if the previous `if` was false.
- else – Run a block when earlier if/elif tests are false.
- match – Choose a branch based on pattern matching.
- case – A single pattern branch inside a match statement.
- for – Repeat a block for each item in a sequence.
- while – Repeat a block while a condition stays true.
- break – Exit the nearest loop immediately.
- continue – Skip to the next loop iteration.
- pass – Do nothing; placeholder where a statement is required.

## Function & Return Keywords
- def – Define a function.
- return – Exit a function and optionally provide a value.
- yield – Produce a value from a generator and pause function execution.
- lambda – Create a small anonymous function.
- global – Declare a name as module-level inside a function.
- nonlocal – Declare a name from an enclosing (non-global) scope.

## Exception Handling Keywords
- try – Start a block that may raise exceptions.
- except – Handle specific exceptions raised in a try block.
- finally – Run a block whether or not an exception occurred.
- raise – Trigger an exception.
- assert – Check a condition and raise an AssertionError if false.

## Class & Object-Oriented Keywords
- class – Define a new class.
- self – Conventional name for the instance parameter in methods; not a Python keyword.
- del – Remove a name or an item from a collection.

## Boolean & Logical Keywords
- and – True if both operands are true.
- or – True if at least one operand is true.
- not – Negate a boolean value.
- is – Test object identity.
- in – Test membership in a container.

## Import & Module Keywords
- import – Bring a module into the current namespace.
- from – Import specific names from a module.
- as – Give an imported name or expression an alias.

## Asynchronous Programming Keywords
- async – Define an asynchronous function or context.
- await – Pause until an awaitable completes inside async code.

## Context Management
- with – Wrap a block with setup and teardown behavior from a context manager.

## Special Constants
- True – Boolean true value.
- False – Boolean false value.
- None – Represents absence of a value.

---

All keywords above match Python's official keyword set; `self` is shown as a common convention, not an actual keyword.
